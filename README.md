# 🧠 A. Cognitive Traits (6 Traits)  

Cognitive traits represent **how the brain processes and understands information.**  

🔥 **These are the foundation of intelligence, reasoning, and conceptual mastery.**  

---

## 1. Pattern Recognition Speed  

The mind’s ability to detect **relationships, similarities, and hidden patterns.**  

### 🚀 High Pattern Recognition  
- Understands math concepts quickly  
- Identifies trends in data  
- Spots logical connections  
- Solves problems faster  

### ⚠️ Low Pattern Recognition  
- Needs step-by-step breakdown  
- Struggles to see the “bigger picture”  

### 📊 Measured Using  
- Speed of correct answers on pattern-based questions  
- Eye movement smoothness during abstract problems  
- Time gap between recognition and response  

---

## 2. Concept Absorption Rate  

How quickly a student converts **new information into understanding.**  

### 🚀 High Absorption  
- Learns quickly  
- Adapts to new topics fast  

### ⚠️ Low Absorption  
- Needs multiple examples  
- Requires repeated explanations  

### 📊 Measured Using  
- First-attempt accuracy  
- Reduction in response time across attempts  
- Confidence change within a new topic  

---

## 3. Abstract Thinking Index  

Ability to **generalize, conceptualize, and think beyond surface-level details.**  

### 🚀 High Abstract Thinking  
- Understands theory easily  
- Strong reasoning skills  
- Excels in math & science  

### ⚠️ Low Abstract Thinking  
- Prefers concrete examples  
- Needs visual support  
- Struggles with theory  

### 📊 Measured Using  
- Performance on “why” questions  
- Eye fixation on abstract symbols  
- Logical vs literal mistake patterns  

---

## 4. Memory Retention Strength  

Ability to **store, retain, and recall information over time.**  

### 🚀 High Retention  
- Strong long-term memory  
- Fewer repetitions needed

## Step 1 — SVG Setup  

```
const width = 600;
const height = 600;

const svg = d3.select("#genome-wheel")
  .append("svg")
  .attr("width", width)
  .attr("height", height);

const g = svg.append("g")
  .attr("transform", `translate(${width/2}, ${height/2})`);
``` 

---

## Step 2 — Arc Generator  

```
const arcGen = d3.arc()
  .innerRadius(120)
  .outerRadius(d => 120 + d.value * 2.5)
  .startAngle(d => d.start)
  .endAngle(d => d.end)
  .cornerRadius(4);
``` 

---

## Step 3 — Angle Mapping  

```
const angleStep = (2 * Math.PI) / 24;
``` 

Each trait:  

- start = i × angleStep  
- end = (i + 1) × angleStep  

---

## Step 4 — Data Binding  

```
g.selectAll("path.trait-arc")
  .data(traits)
  .join("path")
  .attr("class", "trait-arc")
  .transition()
  .duration(800)
  .attr("d", arcGen)
  .attr("fill", d => categoryColor(d.category));
``` 

---

## Step 5 — Tooltips  

```
.on("mouseenter", (event, d) => showTooltip(d))
.on("mouseleave", hideTooltip);
```
